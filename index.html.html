What is Library of Congress Classification (LCC)?

The Library of Congress Classification system is the standard used by most academic and research libraries in the United States to organize their collections. Unlike the Dewey Decimal system (which uses only numbers), LCC uses a combination of letters and numbers to group books by subject.

Every book on the shelf has a call number — a unique address that tells you exactly where that book lives. Your job as a student worker is to read these addresses accurately so you can shelve and retrieve books with confidence.

.S62 Cutter

Each element is read in order from top to bottom on the spine label, and the entire call number is read left to right across the shelf.

⚖️ The Five Sorting Rules

📌 Rule 1 — Letters First (Alphabetically)

Sort by the letter portion of the call number first. Single letters come before double letters; double before triple.

  • B → BC → BD → BF
  • H → HA → HB → HD
  • P → PA → PE → PQ → PR

🔢 Rule 2 — Class Number as a Whole Number

After the letters, sort by the class number. Read it as a whole number (integers before decimals).

  • PR 5 comes before PR 25
  • QA 39 comes before QA 300
  • HB 1 comes before HB 171

🔢 Rule 3 — Class Number Decimals

When the class number has a decimal, it is treated as a true decimal. This is the same as the Cutter rule.

LB 2395.7 → comes before → LB 2396

2395.7 is less than 2396, so it comes first on the shelf.

⭐ Rule 4 — The Cutter (Decimal Rule!)

This is the Golden Rule and the most common mistake. The digits after the Cutter letter are read as a decimal fraction, NOT a whole number.

.B33
= B + 0.33
→ BEFORE →
.B4
= B + 0.4

Why? Because 0.33 < 0.4. Think of it as money: $0.33 is less than $0.40.

  • .A2 → .A25 → .A3 → .A33 → .A4
  • .B125 → .B13 → .B2 → .B234 → .B25

🗓️ Rule 5 — "Nothing Before Something"

When two call numbers are identical except one has an additional element (a second Cutter, a volume, a year, or a copy number), the shorter one comes first.

  • No date → older date → newer date
  • No second Cutter → with second Cutter
  • HQ 1206 .R63 before HQ 1206 .R63 A3
  • QA 76 .H45 2010 before QA 76 .H45 2015

🔤 Rule 6 — Second Cutters & Work Marks

Some call numbers have a second Cutter (for title or editor). After matching the first Cutter, sort alphabetically by the second Cutter, again treating its digits as decimals.

  • .R63 A3 before .R63 B45
  • .H45 T87 2000 before .H45 T87 2012
